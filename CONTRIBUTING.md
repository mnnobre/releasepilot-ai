# Contributing

ReleasePilot AI is currently a portfolio product under active development.
Issues and focused pull requests are welcome.

## Local workflow

```bash
npm install
npm run dev
```

Before opening a pull request:

```bash
npm run lint
npm test
npm run build
```

Keep changes small, explain the user problem, and include tests for behavior.
Demo data must remain clearly identified and must not contain real customer
information.
